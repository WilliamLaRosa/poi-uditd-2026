Cargando datos desde datos.json…

⚠ No se pudo cargar datos.json

El dashboard necesita ejecutarse desde un servidor local. No funciona con doble clic porque el navegador bloquea la lectura de archivos locales por seguridad.

Soluciones rápidas:

Opción A — Python (recomendado):
  1. Abre CMD en la carpeta del proyecto
  2. Escribe: python -m http.server 8080
  3. Abre: http://localhost:8080

Opción B — VS Code:
  Instala extensión "Live Server" → clic derecho en index.html → Open with Live Server
